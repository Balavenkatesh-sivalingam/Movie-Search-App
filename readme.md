# Movie Search App

A responsive React application that allows users to search, filter, and view detailed information about movies using the [OMDB API](https://www.omdbapi.com/).  

This project demonstrates practical use of React, React Router, Tailwind CSS, and API integration.

---

## Features

- **Search Movies**: Enter a movie title or keyword to search for movies.
- **Filter by Type**: Filter results by movie type (movie, series, episode) using the API.
- **Pagination**: Navigate through large sets of search results.
- **Movie Details Page**: View detailed information including poster, title, year, genre, plot, ratings, cast, and runtime.
- **Responsive Design**: Fully responsive layout suitable for desktop and mobile devices.
- **Navigation**: Implemented using React Router.
- **Loading & Error Handling**: Loading indicators during API calls and user-friendly messages for errors or no results.

---

## Tech Stack

- **Frontend**: React.js, Tailwind CSS  
- **Routing**: React Router  
- **State Management**: useState, useEffect, URL search parameters  
- **API**: [OMDB API](https://www.omdbapi.com/)
